## 4주차 과제
단어를 빨리 외울수있게해주는 앱
외국어를 공부하는 사람들에게 단어를 빨리 외울 수 있게 해주는 앱이다.
단어를 외우다보면 너무 시간이 오래걸려서 시간효율이 떨어지기때문에
타이머를 맞춰서 시간내에 단어를 외울수있게 활용한다.
자신의 통장을 자동이체 설정을 하고 시간내에 목표치에 도달하지 못하면 특정금액씩 빠져나가서
목표치에 도달할시에만 금액을 되찾을수있게 해놓는 앱이다.
